## Keyboard Shortcuts

- `D` - Toggle Dark Mode
- `S` - Open Settings
- `F` - Toggle Fullscreen
- `1` - Clock Mode
- `2` - Stopwatch Mode
- `3` - Timer Mode
- `4` - Pomodoro Mode
- `5` - World Clock Mode
- `ESC` - Close Modals

## Deployment

This project is deployed on GitHub Pages. To enable GitHub Pages:

1. Go to your repository settings on GitHub
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the branch (usually `main` or `master`)
4. Select the folder (usually `/ (root)`)
5. Click "Save"
6. Your site will be available at: `https://himanshumodi3108.github.io/Digital-Clock/`

The site will automatically update whenever you push changes to the selected branch.
